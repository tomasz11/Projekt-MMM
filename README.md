# Projekt-MMM
Projekt nr 10. Dany jest układ RLC.gdzie R, L, C to parametry modelu. Należy zaimplementować symulator tego układu umożliwiając uzyskanie odpowiedzi czasowych układu na pobudzenie przynajmniej trzema rodzajami sygnałów wejściowych.Należy wykreślić charakterystyki częstotliwościowe Bodego (amplitudową i fazową) oraz odpowiedź układu.
